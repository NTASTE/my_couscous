榜单接口【action=board&id=*】

获取某榜单应用列表

请求说明

URL

http://m.baidu.com/api?action=board&id=*

请求方式

GET

支持格式

JSON/XML (默认XML)

请求参数

字段名称 备注 是否必选
from 来源，合作方标识，由百度BD提供 必选
token 标记，由百度分配的唯一的token，用于验证请求方的合法性 必选
type 服务类别，type=app，对于android，type的值为app 必选
id 榜单编码。参见 数据接口》榜单列表 必选
bdi_imei android手机imei号。参见 附录2》数据加密 必选
bdi_loc 客户端所在的城市 例：北京市。（base64处理） 必选
bdi_uip 客户端ip（客户端用户请求ip） 必选
bdi_bear 网络环境（WF/4G/3G/2G） 必选
resolution 手机分辨率 例：720_1280 必选
dpi 屏幕密度 例：320 必选
apilevel 系统api level 例：18（对应android 4.3） 必选
os_version 系统版本号 例：4.3 必选
brand 手机品牌 例：OPSSON 必选
model 手机型号 例：Q1 必选
pver 协议版本（此版本固定为3） 必选
sign 签名，只针对该表的参数签名 参见 附录3》sign生成算法 必选
uid 用户唯一标识 可选
bdi_mac 无线网卡mac地址 可选
bdi_cid 基站id 可选
bdi_imsi android手机imsi号。（base64处理） 可选
rn 每页显示结果数，默认rn=10，最大值为20 可选
pn 偏移量，默认pn=0，即第一页，必须为rn的整数倍 可选
ct 客户端点击时间 可选
cname 客户端(宿主)名称 可选
cver 客户端(宿主)版本号 可选
cpack 客户端(宿主)包名 可选

注意

为更好进行机型和网络适配，对于所有请求参数，尽可能携带可选参数; 参数中包含除字母、数字的字符，都需要进行urlencode
